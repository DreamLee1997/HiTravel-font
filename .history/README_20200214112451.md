## 项目运行（nodejs 6.0+）
``` bash
# 克隆到本地
git clone git@github.com:DreamLee1997/HiTravel-font.git

# 安装依赖
npm install

# 开启本地服务器localhost:8080
npm run dev

# 发布环境
npm run build
```
